# IoT Weather Station

## Objective
To monitor temperature and humidity using ESP32 and DHT22 sensor.

## Components Used
- ESP32
- DHT22 Sensor
- Wokwi Simulator
- MicroPython

## Circuit Connections

| DHT22 | ESP32 |
|--------|--------|
| VCC | 3V3 |
| SDA | GPIO 15 |
| GND | GND |

## Working

The DHT22 sensor measures temperature and humidity. The ESP32 reads the sensor data and displays it on the serial monitor every 2 seconds.

## Output

The system successfully displays real-time temperature and humidity readings.
